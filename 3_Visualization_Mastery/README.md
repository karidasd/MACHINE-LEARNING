# 🎨 Μάθημα 3: Η Τέχνη της Οπτικοποίησης

Σε αυτό το μάθημα θα μάθουμε πώς να επιλέγουμε το σωστό γράφημα για τα δεδομένα μας.

### 📊 Τύποι Γραφημάτων:
* **Line Charts**: Για να δείξουμε την εξέλιξη στο χρόνο (π.χ. τιμή μετοχής).
* **Bar Charts**: Για συγκρίσεις μεταξύ κατηγοριών.
* **Histograms**: Για να δούμε πού "μαζεύονται" οι τιμές μας (κατανομή).
* **Scatter Plots**: Για να δούμε αν δύο πράγματα σχετίζονται (π.χ. ύψος και βάρος).

### 🛠️ Βιβλιοθήκες:
Θα χρησιμοποιήσουμε την **Matplotlib** και την **Seaborn** (που κάνει τα γραφήματα να φαίνονται πιο επαγγελματικά).
