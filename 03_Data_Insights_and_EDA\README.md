# 🔍 Μάθημα 3: Εξερευνητική Ανάλυση Δεδομένων (EDA)

Σε αυτό το στάδιο, γινόμαστε "ντετέκτιβ". Δεν φτιάχνουμε απλά γραφήματα, αλλά προσπαθούμε να απαντήσουμε σε δύσκολες ερωτήσεις.

### ❓ Ερωτήματα που θα απαντήσουμε:
* Υπάρχει σχέση (συσχέτιση) μεταξύ δύο μεγεθών;
* Ποιες είναι οι ακραίες τιμές που επηρεάζουν τον μέσο όρο;
* Ποιο είναι το "προφίλ" του καλύτερου πελάτη/προϊόντος μας;

### 🛠️ Νέα Εργαλεία:
* **Correlation Matrix (Heatmaps)**: Για να δούμε ποιες στήλες "συνεργάζονται" μεταξύ τους.
* **Distribution Plots**: Για να δούμε την πυκνότητα των τιμών.
