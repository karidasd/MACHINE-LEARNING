📘 Μάθημα 1: Εισαγωγή στην Ανάλυση Δεδομένων (Data Analysis)
Καλώς ήρθατε στο πρώτο μάθημα! Εδώ θα μάθουμε πώς να μετατρέπουμε "ακατέργαστα" δεδομένα σε χρήσιμες πληροφορίες.

🤔 Τι είναι η Ανάλυση Δεδομένων;
Η Ανάλυση Δεδομένων είναι η διαδικασία κατά την οποία εξετάζουμε, καθαρίζουμε και μοντελοποιούμε δεδομένα με σκοπό:

Την ανακάλυψη χρήσιμων πληροφοριών.

Την εξαγωγή συμπερασμάτων.

Την υποστήριξη στη λήψη αποφάσεων.

🛠️ Τα Εργαλεία μας
Σε αυτό το μάθημα θα χρησιμοποιήσουμε τη γλώσσα προγραμματισμού Python και τις εξής βιβλιοθήκες:

Pandas: Το "Excel" της Python. Χρησιμοποιείται για τη διαχείριση πινάκων.

NumPy: Για γρήγορους μαθηματικούς υπολογισμούς.

Matplotlib / Seaborn: Για τη δημιουργία γραφημάτων και οπτικοποίηση.

🚀 Η Διαδικασία (Workflow)
Κάθε αναλυτής ακολουθεί αυτά τα βήματα:

Data Collection: Από πού προέρχονται τα δεδομένα; (π.χ. CSV, SQL, Web Scraping).

Data Cleaning: Διόρθωση λαθών και διαγραφή κενών τιμών.

Exploratory Data Analysis (EDA): Περιγραφική στατιστική (Μέσος όρος, μέγιστη/ελάχιστη τιμή).

Visualization: Μετατροπή των αριθμών σε εικόνες (Bar charts, Scatter plots).

⌨️ Οι πρώτες σας εντολές
Ανοίξτε το τερματικό σας και βεβαιωθείτε ότι έχετε τα απαραίτητα εργαλεία:

pip install pandas numpy matplotlib seaborn
